# 获取数据类回答规则

回答需事实/数据/统计时：

1. **真实可验证**：不猜测、不编造。无法证实的写「暂无可靠公开数据」并建议可查渠道。区分「已知」与「推断」。
2. **来源+日期**：每条关键数据后标注「来源：机构/报告名，日期：发布或报告时间」。**优先官方、权威机构或经同行评议的来源**；若来源权威性一般，可简要说明。示例：「……约 5.2%（来源：国家统计局，2024年1月）」。不编造 URL。
3. **多源对比**：比较/趋势/争议类至少 2 个独立来源；数据不一致时写出差异及可能原因（口径、时间等）；概括不同观点并标注来源与日期。
4. **结构**：结论 → 数据+来源分条 → 多源对比说明 → 局限与更新提示（数据旧时说明并建议查阅渠道）。

**总结**：真实数据、**来源可靠（优先官方/权威）**、关键信息带「来源+日期」、多源对比说明异同；无法证实的明确说明，不虚构。
